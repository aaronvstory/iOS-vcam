 RTMP over USB on iOS
A detailed guide with step-by-step explanations

üéØ Why is this needed?
You want to stream video from your computer to your iPhone, but WiFi is slow or unstable. The solution is to transfer via a USB cable, like on Android with adb reverse. On iOS, it's a bit more complex, but achievable.

What we will get in the end:

Stable RTMP stream without WiFi
Minimal latency
Works via a standard USB cable

3
Forward SSH port over USB
3.1. In the same command prompt window, enter:
iproxy.exe 2222 22
Press Enter.

What this command means:
iproxy.exe - run the program
2222 - local port on the computer
22 - SSH port on the iPhone
Basically: "everything that comes to port 2222 on the computer, send to port 22 on the iPhone"
What should happen:
The window will show text:
waiting for connection

This is normal! The program is running and waiting for a connection.
üì∑Screenshot
–û–∫–Ω–æ MonaServer
IMPORTANT: Do not close this window! It must be running the entire time you are streaming.
4
Create a reverse tunnel via SSH
4.1. Open a NEW command prompt window
Without closing the previous window with iProxy:

Press Win + R again, type powershell, and press Enter
OR press Win + X and select Windows PowerShell
4.2. Connect to the iPhone via SSH
Enter the command:

ssh -R 1935:localhost:1935 mobile@localhost -p 2222
Press Enter.

Command breakdown:
ssh - connect via SSH
-R 1935:localhost:1935 - create a reverse tunnel for port 1935
mobile - username on the iPhone
@localhost - connect to the local address (via iproxy it goes to the iPhone)
-p 2222 - use port 2222 (which we forwarded in the previous step)
4.3. First connection
On the first connection, a question will appear:

Are you sure you want to continue connecting (yes/no/[fingerprint])?
Type: yes and press Enter.

4.4. Enter the password
A prompt will appear:

mobile@localhost's password:
Enter: the password you specified when installing the jailbreak and press Enter.

Note: When you type the password, the characters are not displayed - this is normal for security. Just type and press Enter.
4.5. Shell configuration (first time)
The Z Shell configuration menu may appear:

This is the Z Shell configuration function for new users...
--- Type one of the keys in parentheses ---
Just press: 0 (zero) and Enter.

What should happen:
You will see the iPhone command prompt, for example:
iPhone:~ mobile$

This means you are inside the iPhone! The SSH tunnel is working.

üîç What if it doesn't work?
Problem 1: "idevice_id.exe not found"
Solution: You did not navigate to the iProxy folder.

Use the cd command to go to the folder
Or provide the full path: C:\iproxy\idevice_id.exe -l
Problem 2: iPhone is not detected
Solution:

Install iTunes (even if you don't use it)
Check that you tapped Trust on the iPhone
Try a different USB cable/port
Reboot the iPhone
Problem 3: SSH does not connect
Solution:

Make sure OpenSSH Server is installed on the iPhone
Check that the window with iproxy.exe 2222 22 is still running
Try connecting as root instead of mobile:
ssh -R 1935:localhost:1935 root@localhost -p 2222
Problem 4: Video does not play on iPhone
Solution:

Make sure LaL-Server is running on the computer
Check that both windows (iProxy and SSH) are running
Make sure the stream is started in OBS
Check the address in the player: rtmp://127.0.0.1:1935/live/test
